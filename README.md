RoadWatch AI

AI-powered road monitoring system that detects potholes, garbage, and damaged roads from uploaded images to help improve road safety and cleanliness.


Overview

RoadWatch AI is a smart web-based application developed for hackathons and learning purposes.
The system analyzes road images using AI/ML techniques and identifies issues like:
 Potholes
 Garbage on roads
 Damaged road areas

The project aims to support smarter city maintenance and faster issue reporting.


How It Works
User uploads a road image
The system processes the image
AI model detects road issues
Result is displayed to the user


Purpose
This project was built to explore how AI can help solve real-world urban problems related to road maintenance and cleanliness.
